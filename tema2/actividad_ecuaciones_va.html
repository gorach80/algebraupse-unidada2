Inecuaciones con Valor Absoluto

El valor absoluto actúa como un operador de distancia geométrica (métrica) en el espacio euclidiano de una dimensión \(\mathbb{R}\). Las desigualdades con valor absoluto se resuelven analíticamente mediante la equivalencia lógica a inecuaciones simultáneas o disyuntivas.

Caso 1: Vecindad Acotada (Menor o igual que)

Sea \(x \in \mathbb{R}\) y sean \(c, d \in \mathbb{R}\) con \(d > 0\). La inecuación \(|x - c| \le d\) representa el conjunto de puntos cuya distancia a \(c\) no supera el valor \(d\). Formalmente:

\[|x - c| \le d \iff -d \le x - c \le d \iff c - d \le x \le c + d\]

El conjunto solución es el intervalo compacto (cerrado y acotado): \(S = [c - d, \, c + d]\).

Demostración formal: Por la definición analítica del valor absoluto: - Si \(x - c \ge 0\), entonces \(|x - c| = x - c\). La desigualdad \(|x - c| \le d\) se escribe como \(x - c \le d\). Combinando ambas, obtenemos \(0 \le x - c \le d\). - Si \(x - c < 0\), entonces \(|x - c| = -(x - c)\). La desigualdad \(|x - c| \le d\) equivale a \(-(x - c) \le d \iff x - c \ge -d\). Combinando ambas, tenemos \(-d \le x - c < 0\). Al realizar la unión disyunta de ambos subconjuntos, se unifica en la doble inecuación \(-d \le x - c \le d\), que al sumar \(c\) resulta en \(c - d \le x \le c + d\).

Caso 2: Vecindad Excluida (Mayor o igual que)

La inecuación \(|x - c| \ge d\) representa todos los puntos cuya distancia al centro \(c\) es al menos \(d\). Su modelado formal corresponde a una disyunción lógica:

\[|x - c| \ge d \iff (x - c \le -d) \lor (x - c \ge d) \iff x \in (-\infty, c - d] \cup [c + d, \infty)\]

El conjunto solución corresponde al complemento del intervalo abierto central: \(S = \mathbb{R} \setminus (c - d, \, c + d)\), formado por la unión de dos intervalos no acotados (rayos infinitos).

Caso General con Coeficiente Lineal: \(|ax + b| \le c\)

Para resolver \(|ax + b| \le c\) con \(a, b, c \in \mathbb{R}\) y \(c > 0\), aplicamos la propiedad del Caso 1 sobre el término lineal \(ax+b\):

\[|ax + b| \le c \iff -c \le ax + b \le c \iff -c - b \le ax \le c - b\] Si \(a > 0\), el intervalo solución es: \[\left[\frac{-c - b}{a}, \, \frac{c - b}{a}\right]\] Si \(a < 0\), al dividir por \(a\) los sentidos de las desigualdades se invierten, resultando en: \[\left[\frac{c - b}{a}, \, \frac{-c - b}{a}\right]\]

Autoevaluación Aplicada (Ingeniería Industrial)

Resuelve los siguientes problemas contextualizados en procesos de manufactura y control de calidad industrial.

1. Control de Calidad y Tolerancia Dimensional
En una línea de ensamblaje automatizada de la UPSE, el diámetro nominal de un pasador cilíndrico de acero debe ser de \(12.00\text{ mm}\). El departamento de control de calidad establece que la desviación absoluta tolerable del diámetro \(x\) no debe superar los \(0.05\text{ mm}\). Expresa esta tolerancia como inecuación y determina el intervalo admisible.
2. Rango de Operación Térmica Segura
El reactor químico de una refinería opera de manera estable a una temperatura promedio de \(150^\circ\text{C}\). Por motivos de seguridad industrial, el sistema de alarma de emergencia se activa si la temperatura \(T\) varía en más de \(15^\circ\text{C}\) de su valor promedio. Determina la inecuación formal.
3. Control de Peso de Empaque de Fertilizantes
Un saco de fertilizante producido por una empresa de la península de Santa Elena debe pesar comercialmente \(45.0\text{ kg}\). Si el peso real es \(w\), la inecuación de control de calidad es \(|w - 45.0| \le 0.3\). ¿Cuál de los siguientes pesos del saco está fuera del estándar y por tanto debe ser rechazado por la línea?
4. Modelado de Margen de Ganancias
Un analista de producción industrial determina que la ganancia neta diaria \(G\) en miles de dólares (USD) de una planta de plástico varía según las fluctuaciones de la materia prima según la inecuación \(|G - 8| \le 3\). Encuentra el rango de ganancias diarias.

División Sintética (Regla de Ruffini)

La Regla de Ruffini (división sintética) es un algoritmo derivado del teorema de división euclidiana en el anillo de polinomios \(\mathbb{R}[x]\). Permite dividir de forma óptima un polinomio dividendo \(P(x)\) entre un binomio de primer grado mónico de la forma \(x - r\), donde \(r \in \mathbb{R}\).

Fundamento Teórico y Algoritmo Recursivo

Sean \(P(x) \in \mathbb{R}[x]\) con \(\text{gr}(P) = n \ge 1\) y \(D(x) = x - r\) con \(r \in \mathbb{R}\). El algoritmo garantiza la existencia de polinomios únicos de cociente \(Q(x)\) y residuo constante \(R \in \mathbb{R}\) tales que:

\[P(x) = (x - r)Q(x) + R \quad \text{con } \text{gr}(Q) = n-1 \quad \text{y } R = P(r) \quad \text{(Teorema del Residuo)}\]

Demostración formal del Teorema del Residuo: Evaluando la ecuación fundamental en el punto singular \(x = r\), tenemos: \[P(r) = (r - r)Q(r) + R = 0 \cdot Q(r) + R = R\] De aquí se deriva el Teorema del Factor: \((x - r)\) es un factor divisor de \(P(x)\) si y solo si \(P(r) = 0\) (es decir, \(r\) es raíz de \(P(x)\)).

Relaciones de Recurrencia para Polinomios Cúbicos: Sea \(P(x) = a_3x^3 + a_2x^2 + a_1x + a_0\). Para el cociente \(Q(x) = q_2x^2 + q_1x + q_0\) y residuo \(R\), el algoritmo de Ruffini establece la siguiente secuencia recursiva: \[q_2 = a_3, \quad q_{k-1} = a_k + r \cdot q_k \quad (\text{para } k = 2, 1), \quad R = a_0 + r \cdot q_0\]

Métrica de la Recta Real: Valor Absoluto

El valor absoluto constituye el operador estándar de distancia (métrica euclidiana de dimensión 1) sobre el cuerpo de los números reales \(\mathbb{R}\). La inecuación de valor absoluto \(|x - c| \le d\) delimita rigurosamente una vecindad cerrada.

Definición Formal de Métrica y Vecindades

El par \((\mathbb{R}, d)\) es un espacio métrico donde la función de distancia \(d: \mathbb{R} \times \mathbb{R} \to [0, \infty)\) se define por:

\[d(x, y) = |x - y| = \sqrt{(x - y)^2}\]

Esta función cumple rigurosamente las propiedades de no negatividad (\(d(x,y)\ge 0\)), identidad de los indiscernibles (\(d(x,y)=0 \iff x=y\)), simetría (\(d(x,y)=d(y,x)\)) y la desigualdad triangular (\(d(x,z) \le d(x,y) + d(y,z)\)).

Estructura de Entorno Cerrado (Vecindad Compacta): Dada una coordenada central \(c \in \mathbb{R}\) y una tolerancia radial \(d \in \mathbb{R}^+\), la vecindad de radio \(d\) en torno a \(c\) es el conjunto: \[V_d(c) = \{x \in \mathbb{R} \mid d(x, c) \le d\} = \{x \in \mathbb{R} \mid |x - c| \le d\}\] Por definición analítica del valor absoluto, esta condición se traduce en la inecuación lineal doble: \[|x - c| \le d \iff -d \le x - c \le d \iff c - d \le x \le c + d \iff x \in [c - d, \, c + d]\]

Glosario y Lenguaje Formal

Definiciones fundamentales y estructuras algebraicas formales que rigen los conceptos de expresiones polinómicas y métricas del valor absoluto.

Expresión Algebraica

Sea \(V = \{x_1, \dots, x_n\}\) un conjunto de variables y sea \(\mathbb{R}\) el cuerpo de los números reales. Una expresión algebraica es una combinación finita de elementos de \(V\) y \(\mathbb{R}\) a través de operaciones de adición, multiplicación, división, potenciación y radicación.

Clasificación estructural:
1. Racional Entera (Polinomio): Elementos pertenecientes al anillo polinómico \(\mathbb{R}[x_1, \dots, x_n]\) (sin variables bajo radicales ni en denominadores).
2. Racional Fraccionaria: Expresiones con variables de exponente entero negativo, representadas en el cuerpo de cocientes \(\mathbb{R}(x_1, \dots, x_n) \setminus \mathbb{R}[x_1, \dots, x_n]\).
3. Irracional: Expresiones con variables bajo radicación (exponentes racionales no enteros).

Grado de un Polinomio (Análisis Polinómico)

Para un monomio \(M = c \cdot x_1^{e_1}x_2^{e_2}\dots x_n^{e_n}\) con \(c \neq 0\):

- **Grado Relativo (GR)** con respecto a \(x_i\) es su respectivo exponente: \(\text{GR}_{x_i}(M) = e_i\).
- **Grado Absoluto (GA)** es la suma de sus exponentes algebraicos: \(\text{GA}(M) = \sum_{j=1}^{n} e_j\).
Para un polinomio \(P = \sum M_k\), el Grado Absoluto corresponde al máximo grado absoluto de todos sus monomios constituyentes: \(\text{GA}(P) = \max_k (\text{GA}(M_k))\).

Valor Absoluto (Leyes Algebraicas)

Para todo \(x \in \mathbb{R}\), se define formalmente mediante la raíz del cuadrado de su valor: \(|x| = \sqrt{x^2}\). Satisface los siguientes enunciados teóricos indispensables:

\[|x| = \begin{cases} x & \text{si } x \ge 0 \\ -x & \text{si } x < 0 \end{cases}\] 1. Multiplicatividad: \(\forall x, y \in \mathbb{R}, \, |x \cdot y| = |x| \cdot |y|\).
2. Desigualdad Triangular: \(\forall x, y \in \mathbb{R}, \, |x + y| \le |x| + |y|\).
3. Desigualdad Triangular Inversa: \(\forall x, y \in \mathbb{R}, \, ||x| - |y|| \le |x - y|\).
